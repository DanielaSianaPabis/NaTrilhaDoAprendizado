<table>

Esta tag define uma tabela HTML

Sintaxe:<table> tabela </table>

Além disso, esta tag dispõe de outros elementos essenciais que auxiliarão a compor uma tabela propriamente dita, são eles:

<tr> linha </tr> - Define uma linha

<th> coluna </th> - Define uma coluna

<td> dados </td> - Define uma célula da tabela (dados inseridos)

Sintaxe:<ins> inserir </ins>
