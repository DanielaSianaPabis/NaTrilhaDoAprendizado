Definição

O CSS (Cascading Style Sheets), que significa Folhas de Estilo em Cascata, tem por finalidade fazer a estilização do conteúdo de um documento html, ou seja, define cores, fontes, espaçamentos, dentre outros, para uma melhor visualização do conteúdo.

Formas de usar o CSS

Inline:

  • Os códigos em CSS são definidos dentro de uma tag HTML específica
  • Afetam apenas a tag em especifico que foi definida
  • Exemplo: Afeta apenas um subtítulo (h2) que foi selecionado

Incorporado:

  • Os códigos em CSS são definidos no cabeçalho, dentro da tag head
  • Afetam apenas o documento em que foi definido
  • Exemplo: Afeta todos os subtítulos (h2) de apenas um documento

Externo:

  • Os códigos em CSS são definidos em um documento externo
  • Afetam todos os documentos que estão linkados ao CSS
  • Exemplo: Afeta todos subtítulos (h2) de todos os documentos que estão linkados ao CSS

Browser Preview e o CSS

Pode-se realizar a instalação da extensão Browser preview, afim de realizar a programação de um documento em um lado da tela e ver de forma simultânea como este está ficando no navegador. Para isso, deve seguir os passos abaixo:

  1. Clique nesta opção
  2. Digite o nome da extensão e clique na opção demarcada pela flecha
  3. Acesse o Live Server e copie o endereço do seu documento HTML
  4. Clique nesta opção para abrir a segunda janela e cole o endereço copiado

Envelopamento

Permite que você selecione uma parte do seu documento ainda não definida com uma tag específica, ou seja, uma parte avulsa, e a envelope de modo a inseri-la em uma tag. Para realizar esta ação siga os passos abaixo:

  1. Selecione a parte desejada;
  2. Precione CTRL+SHIFT+P;
  3. Selecione a opção "Quebrar Linha com Abreviação;"
  4. Digite a tag que deseja envolver na parte que foi selecionada, como: "h1" "p", etc;
  5. Clique em ENTER no seu teclado para efetuar a ação.

Comandos de estilização

Color: cor das letras

Background-color: cor de fundo

Text-align: alinhar texto

Font-family: tipo de fonte

Font-size: regular o tamanho da fonte em pixels
