Box Model

O modelo das caixas em CSS, descreve os boxes geradas pelos elementos HTML detalhando as opções de ajuste de margens, bordas, padding e conteúdo para cada elemento. Entender o Box Model é fundamental para construção de layout em sites.

Posicionamento

Estático - permanece na posição em que foi definido
Relativo - posicionado em relação onde foi definido
Absoluto - pode ser posicionado em qualquer lugar do documento
Fixed - permanece fixo durante a rolagem do conteúdo
Sticky - permanece fixo durante a rolagem a partir de onde foi definido

Float

O float (flutuar) é uma propriedade CSS que tem como função de posicionar e formatar um conteúdo, determinando como este irá flutuar no navegador.

O float pode vir a possuir um dos seguintes valores:

  1. left - O elemento permanece flutuando do lado esquerdo do navegador.
  2. right - O elemento permanece flutuando do lado direito do navegador.
  3. none - O elemento não flutuará. Permanece exibido no modelo padrão.
  4. inherit - O elemento herda o valor flutuante do pai.

Right:

None:

Note por fim, que esta imagem genérica de girassóis está demarcada como "none", portanto, não haverá flutuação e este texto que está sendo digitado agora não ficará alinhado a imagem como nos outros dois exemplos de float acima (left e right). Assim, percebe-se a aplicação da propriedade float quando deixa uma imagem sem quaisquer flutuações, ou seja, permanece no modelo padrão.
