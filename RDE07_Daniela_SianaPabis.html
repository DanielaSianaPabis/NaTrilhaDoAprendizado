BRANCHES

Branches: Definição

Comportam-se como ramificações de um projeto de software ou documento que tem por finalidade o desenvolvimento paralelo de:

  • Novas funcionalidades;
  • Correção de problemas;
  • Alterações de código.

Branches: Repositório

Branch Master ou Main - braço primário (principal) de um projeto em um repositório.

Branch Secundária - braço secundário de um projeto em um repositório e é onde se realizam alterações no repositório de forma paralela a Branch Master, sem afetar os usuários caso algum erro seja feito.
