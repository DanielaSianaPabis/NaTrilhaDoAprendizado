TAGS

As tags servem para realizar a marcação de conteúdo , onde cada uma possui uma funcionalidade específica. Sem elas, o navegador exibiria um texto plano.

Sintaxe geral das tags: <nome_tag> conteúdo </nome_tag>

Ainda, as tags podem dispor de atributos, que apresentarão alguma informação extra, como uma cor, por exemplo.

Sintaxe geral das tags com atributos: <nome_tag atributo="valor"> conteúdo </nome_tag>

TAGS APRESENTADAS EM AULA

<html>

O documento html é definido pela tag html e dentro desta existem outras tags, as quais serão subordinadas ao html.

Sintaxe da tag html:
 <html>
 conteúdo
</html>

<meta>

Esta tag permite que o navegador fique ajustado para o que precisamos, como:

  • <meta charset="UTF-8">
  • <meta name="viewport content="width=device-width, initial-scale=1.0">

<head>

Esta tag define o cabeçalho.

Sintaxe da tag head:
 <head>
 conteúdo
</head>

<title>

Esta tag define uma barra de título para o navegador.

Sintaxe da tag title: <title> conteúdo </title>

<body>

Tag onde o conteúdo será escrito.

Sintaxe da tag body:
 <body>
 conteúdo
</body>

<h1>

Esta tag de heading representa o primeiro nível de importância, sendo utilizada para demarcar títulos principais.

Sintaxe da tag h1: <h1> título 1 </h1>

<h2>

Esta tag de heading representa o segundo nível de importância, sendo utilizada para demarcar subtítulos.

Sintaxe da tag h2: <h2> título 2 </h2>

Tags entre <h3 e h5>

Tags entre <h3> e <h5> reduzem o tamanho das letras a medida que chegam ao h6, sendo utilizados de acordo com o grau de importância almejado.

  • Sintaxe da tag h3: <h3> título 3 </h3>

  • Sintaxe da tag h4: <h4> título 4 </h4>

  • Sintaxe da tag h5: <h5> título 5 </h5>

<h6>

Esta tag heading representa o último nível de importância de um título.

Sintaxe da tag h6: <h6> título 6 </h6>

<p>

Utilizado para definir parágrafos.

Sintaxe da tag p: <p> parágrafo </p>

<br>

Utilizado para realizar quebra de linha em um texto.

Sintaxe da tag br: Esta é uma <br> quebra de texto

<b>

Esta tag é utilizada para representar um texto em negrito

Sintaxe da tag b: <b> texto em negrito </b>

<strong>

Tag utilizada para destacar um texto e apesar de ficar visualmente semelhante ao próprio negrito possui o diferencial de trazer uma importância maior aos elementos destados.

Sintaxe da tag strong: <strong> texto destacado </strong>

<i>

Utilizado para demarcar um texto em itálico.

Sintaxe da tag i: <i> texto em itálico </i>

<em>

Esta tag também é utilizada para demarcar em itálico um trecho de um texto, entretanto este atribui uma ênfase aos elementos demarcados, ou seja, um grau de importância maior.

Sintaxe da tag em: <em> texto com ênfase </em>

<big>

Utiizado para realizar a formatação de um texto, visando ampliar o tamanho da fonte deste.

Sintaxe da tag big: <big> fonte ampliada </big>

<font>

Esta tag tem por função de formatar a fonte de um texto e atribuir a este uma cor almejada.

IMAGENS E LISTAS

IMAGENS

IMAGEM:
Representação visual de algo, podendo ser de uma pessoa, objeto ou paisagem
IMAGEM DIGITAL:
Representação bidimensional de uma imagem, com a utilização de números binários devidamente codificados e a permissão para realizar seu armazenamento, transferência, processamento e impressão ou reprodução

FORMATOS/TIPOS DE IMAGENS

O formato de uma imagem se comporta como um padrão de codificação, que determinará o tipo do arquivo gráfico, como: .BMP, .TIFF, .GIF, .SVG, .ICO, .JPEG, .PNG, etc.

OBSERVAÇÃO:

  • .JPEG – Preserva o fundo da imagem
  • .PNG – Não preserva o fundo da imagem
  • A diferença entre cada extensão de arquivo se dá na compactação do arquivo
  • O tamanho da imagem infere diretamente na velocidade para carregar o site

LICENÇAS E REPOSITÓRIOS DE IMAGENS

Deve-se utilizar imagens creative commons a fim de evitar problemas relacionados a diretos autorais, uma vez que estas são imagens distribuídas gratuitamente. Ou ainda, se pode utilizar repositórios gratuitos para a obtenção de imagens, como: unsplash, Pixabay, Rawpixel, etc.
