TAGS

As tags servem para realizar a marcação de conteúdo , onde cada uma possui uma funcionalidade específica. Sem elas,
o navegador exibiria um texto plano.

Sintaxe geral das tags: <nome_tag> conteúdo </nome_tag>

Ainda, as tags podem dispor de atributos, que apresentarão alguma informação extra, como uma cor, por exemplo.

Sintaxe geral das tags com atributos: <nome_tag atributo="valor"> conteúdo </nome_tag>

TAGS APRESENTADAS EM AULA

<html>

O documento html é definido pela tag html e dentro desta existem outras tags, as quais serão subordinadas ao html.

Sintaxe da tag html:
 <html>
 conteúdo
</html>

<meta>

Esta tag permite que o navegador fique ajustado para o que precisamos, como:
<meta charset="UTF-8">
<meta name="viewport content="width=device-width, initial-scale=1.0">

<head>

Esta tag define o cabeçalho.

Sintaxe da tag head:
 <head>
 conteúdo
</head>

<title>

Esta tag define uma barra de título para o navegador.

Sintaxe da tag title: <title> conteúdo </title>

<body>

Tag onde o conteúdo será escrito.

Sintaxe da tag body:
 <body>
 conteúdo
</body>

<h1>

Esta tag de heading representa o primeiro nível de importância, sendo utilizada para demarcar títulos principais.

Sintaxe da tag h1: <h1> título 1 </h1>

<h2>

Esta tag de heading representa o segundo nível de importância, sendo utilizada para demarcar subtítulos.

Sintaxe da tag h2: <h2> título 2 </h2>

Tags entre <h3 e h5>

Tags entre <h3> e <h5> reduzem o tamanho das letras a medida que chegam ao h6, sendo utilizados de acordo com o
grau de importância almejado.

Sintaxe da tag h3: <h3> título 3 </h3>

Sintaxe da tag h4: <h4> título 4 </h4>

Sintaxe da tag h5: <h5> título 5 </h5>

<h6>

Esta tag heading representa o último nível de importância de um título.

Sintaxe da tag h6: <h6> título 6 </h6>

<p>

Utilizado para definir parágrafos.

Sintaxe da tag p: <p> parágrafo </p>

<br>

Utilizado para realizar quebra de linha em um texto.

Sintaxe da tag br: Esta é uma <br> quebra de texto

<b>

Esta tag é utilizada para representar um texto em negrito.

Sintaxe da tag b: <b> texto em negrito </b>

<strong>

Tag utilizada para destacar um texto e apesar de ficar visualmente semelhante ao próprio negrito possui o diferencial de trazer
uma importância maior aos elementos destados.

Sintaxe da tag strong: <strong> texto destacado </strong><

<i>

Utilizado para demarcar um texto em itálico.

Sintaxe da tag i: <i> texto em itálico </i>

<em>

Esta tag também é utilizada para demarcar em itálico um trecho de um texto, entretanto este atribui uma ênfase aos elementos
demarcados, ou seja, um grau de importância maior.

Sintaxe da tag em: <em> texto com ênfase </em>

<big>

Utiizado para realizar a formatação de um texto, visando ampliar o tamanho da fonte deste.

Sintaxe da tag big: <big> fonte ampliada </big>

<font>

Esta tag tem por função de formatar a fonte de um texto e atribuir a este uma cor almejada.

fonte em vermelho

Página Web e Website

Página Web (Webpage) - se comporta como um documento/arquivo, onde se pode colocar imagens,textos tabalas, apresentações, etc.
Permitindo a navegação de uma página para a outra.

Websites ou Sites - se comportam como uma coleção de páginas web armazenadas (em uma pasta ou subpastas) relacionadas de um
servidor web.

HTML (Hypertext Markup Language)

Criada em 1990, o html é uma Linguagem de Marcação de Hipertexto que se baseia em tags , afim de publicar um conteúdo em hipertexto ,
que unem-se por meio de hyperlinks.
