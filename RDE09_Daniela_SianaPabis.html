Multimídia

Se comporta como um meio de comunicação que combina diferentes formas de conteúdo, tais quais:texto, áudio, imagens, animações ou vídeo em uma única apresentação.

Tag <iframe>

Usada para incorporar outro documento ao documento HTML, como:áudio, vídeo, imagens, demais sites,etc.

Incorporar audio no site

Esta incorporação se faz muito útil para quando se quer criar um podcast de tecnologia, ou seja, embutir audios de determinado assunto em um site que almejamos.

Para fazer a incorporação deve:

  1. Acessar um site que diponibilize músicas, como o SoundCloud.
  2. Na barra de pesquisa, digite uma banda, artista, faixa, etc. de sua preferência.
  3. Depois de encontrar, clique em COMPARTILHAR e após em INCORPORADO.
  4. Em INCORPORADO, irá estar visível o código do som escolhido, o qual deve ser copiado e colado no Visual Studio Code.

Incorporar vídeos ao site

O YouTube é uma das principais plataformas onde são disponibilizados vídeos, uma vez que é a melhor opção para se hospedá-los, ja que muitos são longos e necessitam de um armazenamento maior.

Para realizar a incorporação deve:

  1. Acesse o YouTube.
  2. Selecione o vídeo que deseja.
  3. Depois clique em COMPARTILHAR e em seguida em INCORPORAR.
  4. Copie o link que será disponibilizado em tela e cole em seu código no Visual Studio Code.
